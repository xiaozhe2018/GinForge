# 我开源了一个Go微服务框架，一键生成1000+行代码

## 前言
大家好，我是 xiaozhe。

作为一个Go开发者，我在工作中经常遇到这些烦恼：
- 每次开发新功能都要写大量重复的CRUD代码
- 权限系统太复杂，每个项目都要重新搭建
- 微服务部署配置繁琐，Docker/K8s每次都要从头配置

为了解决这些问题，我花了几个月时间开发了 **GinForge** 这个框架。

今天开源出来，希望能帮到有同样需求的朋友们。

## 核心功能

### 1. 一键生成CRUD（我最满意的功能）

我实现了一个代码生成器，只需要一行命令：

```bash
go run ./cmd/generator gen:crud --table=articles
```

就能自动生成7个文件，共1000+行代码！包括：
- 后端4层架构完整代码
- 前端Vue页面（列表+表单）
- 完整的增删改查功能

（贴代码和截图）

**10分钟完成一个模块的开发，这就是我想要的效率！**

### 2. 我做了一个完整的后台管理系统

因为每个项目都需要后台，所以我直接做了一个：
- Vue3 + TypeScript + Element Plus
- 用户管理、角色管理、权限管理、菜单管理
- RBAC四级权限控制

（贴截图）

**拿来就能用，不用再从头写了！**

### 3. 微服务架构和部署

我把之前项目中积累的经验都整合进来了：
- 8个微服务 + API网关
- Docker + docker-compose配置
- Kubernetes + Istio配置
- Nginx反向代理

（贴架构图）

## 快速开始

我尽量让安装步骤简单：

```bash
git clone https://github.com/xiaozhe2018/GinForge.git
cd GinForge
make run
```

30秒就能启动！
